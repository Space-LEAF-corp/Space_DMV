# Space DMV

The world's first **civilian-safe, community-driven** ceremonial safety system
and organization for the space cars of the future.

This repository contains a fictional, educational web prototype for the Space DMV
portal used in the Space LEAF Corp universe.

## Structure

- `index.html` – main hub for Space DMV
- `css/styles.css` – styling for all pages
- `pages/` – subpages (Orientation, Licensing, Registry)
- `public/` – static assets (logos, images)

## Usage

Open `index.html` in a browser (double-click or via a local server) to explore
the Space DMV prototype.
